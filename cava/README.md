# Configuración de CAVA

[CAVA](https://github.com/karlstav/cava) (Console-based Audio Visualizer for ALSA) es un visualizador de audio minimalista renderizado directamente en la terminal interactiva.

## Acerca de estos archivos

Este directorio almacena los ajustes visuales que definen el funcionamiento de la barra espectral. Aquí se controlan:
- Orígenes del audio de tu sistema de sonido (PulseAudio/PipeWire).
- Filtros para las bajas y frecuencias altas.
- Apariencia (colores base, colores degradados dependiendo de la amplitud).
- Carácter utilizado en la consola para dibujar las barras musicales.
