# Entorno de Neovim

Este directorio aloja la configuración esencial de [Neovim](https://neovim.io/), el editor de texto y plataforma de desarrollo por terminal.

## Detalles

El ecosistema de Neovim ha sido adaptado para agilizar la escritura de código o la alteración de scripts de configuración. Usualmente aquí se declara:
- Sistema administrador de extensiones y plugins.
- Colores para el resaltado de sintaxis o para la interfaz (`colorscheme`).
- Reglas para utilidades LSP (servidores de lenguajes), auto-completados e integración con otras dependencias del ecosistema.
- Asignación personalizada de mapas de atajos de teclado en normal, visual, e insert mode.
