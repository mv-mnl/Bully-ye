# Configuración de Kitty

[Kitty](https://sw.kovidgoyal.net/kitty/) es un emulador de terminal moderno que se apoya en el uso intensivo de la GPU, garantizando rendimiento y capacidades avanzadas.

## Funcionalidad

Los elementos centrales controlados en este directorio dictan la experiencia de uso de tu terminal principal:
- Selección de fuentes (Fonts) y ligaduras.
- Colores primarios de la paleta.
- Estilos visuales como la transparecia de la ventana en composición con tu entorno del sistema ("opacity").
- Atajos de teclado para manejar ventanas o múltiples pestañas dentro del mismo emulador.
